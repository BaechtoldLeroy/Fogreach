# Feature: Diablo 2 Style Controls on Mobile

## Problem
Mobile does not yet have the Diablo 2 style movement and attack controls. Players on mobile are missing the core input experience that desktop already has.

## Goal
Implement D2-style tap-to-move and tap-to-attack controls for mobile, matching the desktop gameplay feel.

## Functional Requirements
- **FR-001**: Tap-to-move: tapping a walkable area moves the player to that position
- **FR-002**: Tap-to-attack: tapping an enemy targets and attacks it
- **FR-003**: Hold-to-move: holding a position makes the player walk continuously toward it
- **FR-004**: Movement and attack feel responsive and match the desktop D2-style experience
- **FR-005**: Touch controls do not conflict with existing mobile UI elements (HUD, menus)
