## Implementation

### Subtask T001 — `js/i18n.js` mit API-Skeleton

**Purpose**: Erstelle das i18n-Modul gemäß `contracts/i18n-api.md`.

**Steps**:
1. Datei `js/i18n.js` neu anlegen.
2. Klassisches IIFE-Modul, das `window.i18n` setzt:
   ```js
   (function () {
     const dicts = { de: {}, en: {} };
     let active = 'de';
     const subscribers = new Set();

     function register(lang, dict) { /* ... */ }
     function t(key, params) { /* ... */ }
     function setLanguage(lang) { /* ... */ }
     function getLanguage() { return active; }
     function onChange(cb) { subscribers.add(cb); return () => subscribers.delete(cb); }

     window.i18n = { register, t, setLanguage, getLanguage, onChange };
   })();
   ```
3. `register(lang, dict)`:
   - Validiert `lang ∈ {'de','en'}` (sonst `console.warn` und Return)
   - `Object.assign(dicts[lang], dict)` — shallow merge, idempotent.
4. Keine externen Dependencies, kein localStorage-Zugriff (das macht persistence.js).

**Files**: `js/i18n.js` (NEU, ~60 Zeilen)

**Validation**: In Browser-Konsole nach Reload `typeof window.i18n.register === 'function'` → `true`.

### Subtask T002 — `js/persistence.js` Language-Wrapper

**Purpose**: Sprache als Feld in bestehender Settings-Persistenz.

**Steps**:
1. `js/persistence.js` lesen, `loadSettings()`/`saveSettings()` finden.
2. Zwei neue Funktionen anhängen + auf `window.persistence` exposen:
   ```js
   function getLanguage() {
     const s = loadSettings() || {};
     const lang = s.language;
     return (lang === 'de' || lang === 'en') ? lang : 'de';
   }
   function setLanguage(lang) {
     if (lang !== 'de' && lang !== 'en') {
       console.warn('[persistence] invalid language:', lang);
       return;
     }
     const s = loadSettings() || {};
     s.language = lang;
     saveSettings(s);
   }
   ```
3. Existierende `window.persistence`-Exporte um `getLanguage`, `setLanguage` ergänzen.

**Files**: `js/persistence.js` (modify)

**Validation**:
- Leerer localStorage → `persistence.getLanguage() === 'de'`
- `persistence.setLanguage('en')` → in DevTools Application-Tab: `demonfall_settings_v1.language === 'en'`
- `persistence.setLanguage('fr')` → Warning, kein Schreibvorgang

### Subtask T004 — `setLanguage` Validation

**Purpose**: Robust gegen Garbage-Input.

**Steps**:
1. In `js/i18n.js` `setLanguage(lang)`:
   ```js
   function setLanguage(lang) {
     if (lang !== 'de' && lang !== 'en') {
       console.warn('[i18n] invalid language, falling back to de:', lang);
       lang = 'de';
     }
     if (lang === active) return;
     active = lang;
     subscribers.forEach((cb) => { try { cb(lang); } catch (e) { console.error(e); } });
   }
   ```
2. Subscriber-Aufrufe in try/catch — ein bösartiger Subscriber soll andere nicht abreißen.

**Files**: `js/i18n.js` (modify)

**Validation**:
- `i18n.setLanguage('xx')` → Console-Warning, `i18n.getLanguage() === 'de'`.
- `i18n.setLanguage('de')` während aktiv `'de'` → kein Subscriber-Call (weil `active === lang` Early-Return).

### Subtask T005 — `t()` Fallback-Kaskade

**Purpose**: Robuste Lookup-Logik mit klarer Diagnose bei Lücken.

**Steps**:
1. `t(key, params)`:
   ```js
   function t(key, params) {
     let val = dicts[active] && dicts[active][key];
     if (val === undefined) val = dicts.de && dicts.de[key];
     if (val === undefined) {
       console.warn('[i18n] missing key:', key);
       return '[MISSING:' + key + ']';
     }
     if (params) {
       Object.keys(params).forEach((k) => {
         val = val.split('{' + k + '}').join(String(params[k]));
       });
     }
     return val;
   }
   ```
2. Interpolation: `{name}` wird durch `params.name` ersetzt. Kein Escaping (Phaser-Texte sind plain).
3. Console-Warning ist absichtlich noisy — wir wollen Lücken in Playtests sofort sehen.

**Files**: `js/i18n.js` (modify)

**Validation**:
- `i18n.register('de', { greet: 'Hallo {name}' })` → `i18n.t('greet', {name: 'Welt'})` → `'Hallo Welt'`
- `i18n.t('missing.key')` → `'[MISSING:missing.key]'` + Warning
- Mit aktiver `'en'` und Key nur in `'de'` registriert → DE-Wert (Fallback funktioniert)
