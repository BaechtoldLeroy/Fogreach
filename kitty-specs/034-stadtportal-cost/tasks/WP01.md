# WP01 — Portal Scroll Item & Shop Integration

## Objective

Introduce the portal scroll as a new consumable item tracked in `window.materialCounts.PORTAL_SCROLL`. Add it to Mara's potion tab in the shop, persist it across save/load, and grant the player starting scrolls on a new game.

## Subtasks

### T001 — Add portal scroll to save model & persistence

**Purpose:** Make portal scroll count persist across save/load and give starting scrolls.

**Steps:**
1. Open `js/persistence.js`. Update the `KEYS` object comment to document that the save now includes a `portalScrolls` field.
2. Open `js/storage.js`. Find `saveGame()`.
3. Add `portalScrolls: window.materialCounts?.PORTAL_SCROLL || 0` to the save payload.
4. In `applySaveToState()`, add migration logic:
   ```js
   window.materialCounts = window.materialCounts || {};
   if (typeof s.portalScrolls === 'number') {
     window.materialCounts.PORTAL_SCROLL = s.portalScrolls;
   } else {
     window.materialCounts.PORTAL_SCROLL = window.materialCounts.PORTAL_SCROLL || 2; // starting scrolls for new/old saves
   }
   ```
5. For brand-new games (no save exists), ensure `window.materialCounts.PORTAL_SCROLL` is initialized to 2.

**Files:** `js/persistence.js` (1 comment), `js/storage.js` (~8 lines added)

**Validation:** Save, reload — `window.materialCounts.PORTAL_SCROLL` is preserved. Old save without the field loads with 2 starting scrolls.

### T002 — Add portal scroll to Mara's potion tab

**Purpose:** Let the player buy portal scrolls from the shop.

**Steps:**
1. Open `js/scenes/ShopScene.js`. Find the potion tab item list (the "Tranke" / potions tab).
2. Add a new shop entry for portal scrolls:
   - Name: "Portalrolle" (or "Portal Scroll")
   - Cost: 75 gold (balance TBD, can be adjusted)
   - Effect: increments `window.materialCounts.PORTAL_SCROLL` by 1
   - Icon: reuse an existing scroll-like icon or use a text label
3. Check the player has enough gold via `window.LootSystem.spendGold(cost)` before granting.
4. Refresh the shop UI after purchase to update the player's gold display.

**Files:** `js/scenes/ShopScene.js` (~20-30 lines added)

**Validation:** Open Mara's shop, go to potions tab, see "Portalrolle" listed with cost. Buy one — gold decreases, scroll count increases. Try to buy without enough gold — purchase blocked.

---

### T003 — Starting scrolls for new games

**Purpose:** Ensure new players start with 2 portal scrolls so the mechanic is not punishing early.

**Steps:**
1. Find the new-game initialization code (likely in `js/main.js` or `js/storage.js` where `materialCounts` is first set up).
2. Add `window.materialCounts.PORTAL_SCROLL = 2` to the initialization alongside other starting materials.
3. Verify this does NOT overwrite an existing save's scroll count.

**Files:** `js/storage.js` or `js/main.js` (~2 lines)

**Validation:** Start a fresh game (clear localStorage). Verify `window.materialCounts.PORTAL_SCROLL === 2`.
