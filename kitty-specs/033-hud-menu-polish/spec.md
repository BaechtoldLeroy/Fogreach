# Feature: HUD & Menu Polish

## Problem
Several menus have overlapping UI elements and generally don't look polished. Layout issues make the interface feel unfinished.

## Goal
Fix overlapping elements, improve spacing and alignment, and give menus a more polished, consistent look.

## Functional Requirements
- **FR-001**: Identify and fix all overlapping UI elements across menus
- **FR-002**: Consistent spacing, padding, and alignment across all menu screens
- **FR-003**: Text should not overflow or clip containers
- **FR-004**: Menus should look correct on both desktop and mobile resolutions
- **FR-005**: Visual consistency in fonts, colors, and button styles across all menus
