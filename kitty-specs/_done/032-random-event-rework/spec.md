# Feature: Random Event Rework

## Problem
Random events, especially chest events, are boring and repetitive. The current event pool lacks variety and does not provide interesting choices or surprises.

## Goal
Overhaul the random event system with more diverse, engaging event types beyond simple chest spawns. Events should create memorable moments and meaningful decisions.

## Functional Requirements
- **FR-001**: Expand the event pool beyond chest events (e.g., ambush encounters, NPC encounters, shrine buffs, trap rooms, puzzle elements, cursed items)
- **FR-002**: Events should have varying risk/reward profiles
- **FR-003**: Some events should offer player choices (e.g., take gold or take item, fight or flee)
- **FR-004**: Event frequency and type distribution should feel balanced and not repetitive
- **FR-005**: Chest events remain but are less dominant in the event pool
