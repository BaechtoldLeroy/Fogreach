# WP02 — F11 keyboard shortcut for fullscreen toggle

## Objective

Add an F11 keyboard shortcut that toggles fullscreen during gameplay, preventing the browser's default F11 behavior and keeping the fullscreen state in sync with the settings toggle from WP01.

## Context

WP01 establishes the fullscreen toggle in settings and the persistence layer. This WP adds a keyboard shortcut so players can quickly toggle fullscreen without opening settings. The shortcut must prevent the browser's native F11 (which does not use the Fullscreen API and can cause layout issues with Phaser).

## Subtasks

### T001 — Add F11 key binding in main game scene

**Purpose**: Let players toggle fullscreen with a single keypress during gameplay.

**Steps**:
1. In `js/main.js`, add a `keydown` listener for F11 (`event.key === 'F11'`).
2. Call `event.preventDefault()` to suppress the browser's native fullscreen toggle.
3. If currently in fullscreen (`this.scale.isFullscreen`), call `this.scale.stopFullscreen()`. Otherwise call `this.scale.startFullscreen()`.
4. Update the persisted fullscreen preference in localStorage to stay in sync with the settings toggle from WP01.

**Files**: `js/main.js`.

**Validation**: During gameplay, pressing F11 enters fullscreen. Pressing F11 again exits. The browser's native fullscreen behavior does not interfere.

### T002 — Sync F11 state with settings toggle

**Purpose**: Ensure the settings menu toggle reflects changes made via F11, and vice versa.

**Steps**:
1. Listen for the browser `fullscreenchange` event to detect state changes from any source (F11, settings toggle, Escape key exit).
2. When fullscreen state changes, update the persisted setting in localStorage.
3. When the settings scene opens, read the current `this.scale.isFullscreen` state to set the toggle position.

**Files**: `js/main.js`, `js/scenes/SettingsScene.js`.

**Validation**: Press F11 to enter fullscreen, open settings — toggle shows fullscreen ON. Press Escape to exit fullscreen, open settings — toggle shows fullscreen OFF.

## Acceptance Criteria

- F11 key toggles fullscreen during gameplay.
- Browser default F11 behavior is prevented.
- Fullscreen state from F11 is persisted to localStorage.
- Settings toggle and F11 shortcut stay in sync.
- Pressing Escape to exit fullscreen also updates the persisted state.

## Key Files

- `js/scenes/SettingsScene.js` — Settings toggle sync
