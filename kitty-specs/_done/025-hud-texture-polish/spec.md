# Feature: HUD Texture Polish

## Problem
HUD elements are functional but visually basic — mostly flat rectangles and generated placeholder textures. Lacks the polished look of a finished game.

## Goal
Replace or enhance HUD textures with more detailed, themed graphics matching the dark fantasy setting.

## Functional Requirements
- **FR-001**: Redesign inventory panel (uiPanel) with ornate border/frame
- **FR-002**: Polish ability tile backgrounds with recessed slot look
- **FR-003**: Health bar: stylized heart/blood vial design or ornate frame
- **FR-004**: XP bar: matching ornate frame
- **FR-005**: Item icons (itWeapon, itHead, itBoots, etc.) — improve detail and color depth
- **FR-006**: Consider thematic font for labels
- **FR-007**: Button polish (inventory, settings, mute)
