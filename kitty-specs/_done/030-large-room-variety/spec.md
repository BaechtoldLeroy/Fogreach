# Feature: Large Room Variety

## Problem
Large procedural rooms feel empty and repetitive. There is not enough to do or see in them, and they lack cosmetic variety to make each room feel unique.

## Goal
Make large rooms more interesting by adding interactive objects, environmental details, and cosmetic variations so each room feels distinct and worth exploring.

## Functional Requirements
- **FR-001**: Add decorative props and environmental details (rubble, barrels, crates, pillars, torches, cobwebs, etc.)
- **FR-002**: Add interactive objects (breakable objects, hidden containers, lore items)
- **FR-003**: Cosmetic theme variations per room (overgrown, flooded, crumbling, blood-stained, etc.)
- **FR-004**: Randomized prop placement that respects walkable areas
- **FR-005**: Large rooms should feel noticeably different from each other on repeat visits
