# Feature: Mobile Skill Selection

## Problem
There is currently no way to select skills on mobile. Players cannot switch between abilities during gameplay.

## Goal
Add a mobile-friendly skill selection UI that lets players pick and switch active skills during gameplay.

## Functional Requirements
- **FR-001**: Mobile skill bar with tappable skill slots
- **FR-002**: Skill selection menu accessible during gameplay (not just in town)
- **FR-003**: Active skill highlighted on the skill bar
- **FR-004**: Skill icons with cooldown indicators
- **FR-005**: Skill bar does not obstruct core gameplay area
- **FR-006**: Consistent with desktop skill selection behavior
