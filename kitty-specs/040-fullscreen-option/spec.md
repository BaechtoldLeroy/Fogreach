# Feature: Fullscreen Option

## Problem
There is no fullscreen toggle in the game. Players cannot switch to fullscreen mode for a more immersive experience.

## Goal
Add a fullscreen option to the settings menu that lets players toggle between windowed and fullscreen mode.

## Functional Requirements
- **FR-001**: Fullscreen toggle in the settings menu
- **FR-002**: Toggling fullscreen works on desktop browsers (Fullscreen API)
- **FR-003**: Game canvas scales correctly when entering/exiting fullscreen
- **FR-004**: Fullscreen preference is saved and restored on next session
- **FR-005**: Keyboard shortcut (e.g., F11) to toggle fullscreen during gameplay
